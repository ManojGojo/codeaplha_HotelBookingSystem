Hotel Room Booking System
# Overview

This is a console-based Hotel Room Search and Booking System implemented in Java (Single File) using:

Object-Oriented Programming (OOP)

File I/O for data persistence

Room categorization

Booking & cancellation

Payment simulation

Booking details view

The system allows users to search rooms, book them, cancel reservations, and view booking information.

🚀 Features
# Room Categorization

STANDARD

DELUXE

SUITE

Each room has:

Room ID

Type

Price per night

Availability status

# Search Rooms

Users can search available rooms by category.

# Booking System

Select room

Enter check-in & check-out dates

Simulated payment processing

Booking ID generated automatically

# Cancel Booking

Enter booking ID

Status updated to CANCELLED

Room becomes available again

# View Booking Details

Displays:

Booking ID

User name

Room ID

Check-in date

Check-out date

Booking status

# Technologies Used

Java

OOP (Classes, Enums, Encapsulation)

File Handling (BufferedReader, BufferedWriter)

LocalDate (Java Time API)

Random (Payment Simulation)
